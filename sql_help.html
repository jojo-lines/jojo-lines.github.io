SQL Help

SELECT * FROM dialogue WHERE

Every query you run is automatically prefixed with the above string. If you don't know what it does, don't worry, it's not that important, it just means you have to type less to do a search.

Example queries

speaker = "Yoshikage Kira (8)"

Returns all lines where the speaker is Yoshikage Kira (8).

speaker LIKE "%Hirose%" AND chapter > 2

This returns all lines where the speaker's name contains 'Hirose' and the chapter is greater than 2. The '%' symbol in SQL is just a wildcard, so it means anything could be on either side of the word 'Hirose' and it would still match.

You can also use the OR keyword alongside or instead of the AND keyword to join together complex queries. This should be all you need to know in order to make reasonably sophisticated queries on this database.

For reference, the available row names (e.g., stuff like speaker and chapter) are line_num, volume, volume_total, chapter, chapter_total, speaker, line

Also, don't forget that the table itself has more basic sorting features too, for further narrowing down the results of your query.
